# Loop – Where to add images and videos

Add each file to the path below. Use the **suggested name** so the site picks it up.  
We go **one by one**.

---

## Hero (top of project page)

**1. Hero image or video**  
- **Path:** `public/media/projects/loop/hero/`  
- **Suggested name:** `hero.jpg` (image) or `hero.mov` / `hero.mp4` (video)  
- If you use a video, add a poster (see 2).

**2. Hero poster** (only if hero is a video)  
- **Path:** `public/media/projects/loop/posters/`  
- **Suggested name:** `hero-poster.jpg`  
- Shown while the hero video loads.

## Notes

- **Images:** `.jpg`, `.png`, or `.webp` are fine. Use the suggested base name (e.g. `research.jpg` or `research.png`).
- **Videos:** Use `.mov` or `.mp4`. For section 6 you can use `rhythm-sonic.mov`; the manifest can be updated to `kind: "video"` and optional poster if needed.
- After adding a file, refresh the Loop project page to see it.
